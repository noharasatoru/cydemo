# cydemo

■脱出ボードゲーム（仮題）

■ルール
　①概要
　　ダイスでプレイヤーキャラを移動させ、
　　ゴールを目指す。

　②構成
　　・プレイ人数
　　　２～４人
　　・構成
　　　ゴールライン
　　　ダンジョンエリア(横５マス×縦１２マス)
　　　スタートライン
　　・進行
      ターンプレイヤーから順にダイスを振り、
      出た目が最大移動可能距離となる。
　　　移動範囲距離の回数だけ１枚ずつ裏向きのカードをめくっていく。
      ゴールラインに先にたどり着いた人の勝利。
　　・カード構成（計６０枚）
　　　道×４５
　　　岩×１０
　　　障害×５
　　　※ゲームモードによって使用
　　　（宝箱×）

■対戦形式
　①タイムアタック
　　一番先にゴールエリアにたどり着いたプレイヤーの勝利です。
　②スコアアタック
　　→全プレイヤーゴール時にポイントが一番高いプレイヤーが勝利です
　③バトルロイヤル（案）

■制作仕様
　①Floor タテ１４マス×ヨコ５マス
　②タテ１マス目、１４マス目にはタテ１マス×ヨコ５マスのパネル
　　→タテ１マス目はスタートライン、タテ１４マス目ゴールラインのため
　③タテ２～１３行、ヨコ１～５列にタテ１マス×ヨコ１マスのパネル（計６０枚）配置
　④プレイヤースタート時、スタートラインヨコ１マス～５マス目から初期位置を選択
　⑤ダイス判定
　⑥プレイヤーキャラを移動処理
　　→プレイヤーキャラ周辺に矢印を表示
　　　矢印タップでパネル判断判定
